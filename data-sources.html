8 Biological Databases

GeneXplor aggregates and standardizes data from the world's most trusted genomic and biomedical databases into a single, unified interface.

Source Data Retrieved Integration Approach Endpoint Focus
Ensembl Gene coordinates, transcripts, biotype classification, chromosomal location, strand information. REST API parsing rest.ensembl.org
UniProt Protein info, functional domains, functional descriptions, protein length, gene name aliases. Primary Knowledge Base rest.uniprot.org
ClinVar (NCBI) Clinical variants, pathogenic classifications, disease associations, submission timeline. NCBI E-utilities XML parsing eutils.ncbi.nlm.nih.gov
gnomAD Population allele frequencies (AF), variant consequences, HGVS notation, allele counts. Aggregated local cache / API gnomad.broadinstitute.org
PubMed (NCBI) Research article abstracts, publication dates, journal names, PMC IDs, citation trend timelines. NCBI E-utilities & Summary eutils.ncbi.nlm.nih.gov
AlphaFold DB 3D structural models of proteins, per-residue confidence scores (pLDDT). PDBe Molstar WebGL Integration alphafold.ebi.ac.uk
STRING Protein-protein interaction evidence scores, experimental validations, co-expression scores. REST API JSON queries string-db.org/api
Reactome Categorized biochemical pathways, molecular interactions, signal transduction hierarchies. Content Service API reactome.org/ContentService
